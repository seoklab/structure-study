How It Works

  1. View each target backbone structure and design a sequence for it
  2. Enter your sequences for the problems above
  3. Click submit - you'll be redirected to create a GitHub Issue with your sequences
  4. Our system will automatically process the issue and run AlphaFold3
  5. Results will be posted as a comment on your issue (GitHub will notify you)
  6. Check the leaderboard for rankings across all participants
